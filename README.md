# go-kafka-test

Basic example of using Kafaka in Go.

## Usage.

* Clone this repo
```
git clone https://github.com/woodsaj/go-kafka-test.git
cd go-kafka-test
```

* install Go dependencies
```
go get ./...
```

* start the kafka docker container.
```
./start-kafaka.sh
```

* start the app
```
go run main.go
```
